## Что сделано

### Боковой виджет с каруселью топ-20 серверов

**Компоненты:**
1. **`includes/sidebar_top.php`** — PHP-компонент, серверная отрисовка для SEO
2. **`api/top_servers.php`** — JSON API для обновления данных без перезагрузки
3. **CSS** — фиксированный блок справа, Minecraft-стиль
4. **JS** — карусель с автопрокруткой, кнопки ◀▶, точки-индикаторы, пауза при наведении

**Что показывает каждая карточка:**
- Ранг (#1, #2...)
- Иконка сервера (pixelated)
- Зелёная точка онлайна
- Название (с ⭐ для promoted, ✓ для verified)
- MOTD сервера
- Онлайн (👥 0/2026) или статус оффлайн
- Голоса за месяц (👍)

**Особенности:**
- Promoted серверы — золотая левая полоса
- Highlighted серверы — алмазная левая полоса
- Автопрокрутка каждые 5 сек (настраивается через `SIDEBAR_SLIDE_INTERVAL`)
- Данные обновляются через API каждые 2 мин
- Скрывается на экранах < 1200px
- Не показывается в админке

**Конфигурация (config.php):**
- `SIDEBAR_TOP_COUNT` = 20
- `SIDEBAR_ROTATE_SECONDS` = 120
- `SIDEBAR_SLIDE_INTERVAL` = 5000

**Тесты: 375 — все проходят ✅**
