## Проблемы и решения

| Проблема | Причина | Решение |
|----------|---------|---------|
| Бесконечный редирект HTTPS | Хостинг с прокси, %{HTTPS} всегда off | Добавлена проверка X-Forwarded-Proto |
| Разлогин после смены роли | Роль хранилась в сессии | Роль читается из БД каждый раз |
| Разлогин на хостинге | Привязка сессии к IP, прокси менял IP | Убрана привязка к IP |
| Сервер оффлайн | TCP порт 25560 заблокирован хостингом | Fallback через API mcsrvstat.us |
| Файлы не обновляются | Ручная загрузка на хостинг | Нужно загружать все изменённые файлы |
