## Что сделано

**Этап 2 полностью завершён.** Реализована вся система серверов:

1. **`includes/minecraft_ping.php`** — класс MinecraftPing: полная реализация протокола SLP (Server List Ping). Handshake → Status Request → JSON Response. Парсинг MOTD, favicon, версии. Таймаут 5 сек.
2. **`dashboard/index.php`** — личный кабинет: сводка (серверов, голосов, онлайн), таблица серверов с действиями.
3. **`dashboard/add.php`** — добавление сервера: валидация (IP, порт, лимит 5 шт., дубликаты), автопинг при добавлении, сохранение иконки из favicon, автоопределение версии. Статус pending.
4. **`cron/ping_servers.php`** — cron-скрипт: пинг всех active серверов, запись в server_stats, обновление is_online/players/motd, consecutive_fails, сохранение иконок.
5. **`server.php`** — страница сервера: полная информация, график онлайна (Chart.js, 24ч), голосование (AJAX), отзывы (форма + список), жалоба (модалка), uptime 30 дней, копирование IP.
6. **`servers.php`** — каталог: пагинация, поиск, фильтры (режим, версия, только онлайн), сортировка (голоса, онлайн, рейтинг, новые), promoted вверху.
7. **`index.php`** — главная: топ-10 по голосам, топ-10 по онлайну, новые серверы, общая статистика (серверов, игроков, голосов сегодня), поиск.
