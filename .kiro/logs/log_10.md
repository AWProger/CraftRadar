## Что сделано

### Система верификации владельца

**Логика:**
- Любой пользователь может добавить сервер на мониторинг
- Чтобы получить статус «Владелец» (✓), нужно подтвердить владение
- Для этого генерируется уникальный 6-символьный код (например `7Sl4xK`)
- Владелец добавляет код в MOTD сервера (`server.properties` → `motd`)
- Перезагружает сервер
- Нажимает «Проверить» — система пингует сервер и ищет код в MOTD
- При успехе — пользователь становится владельцем, сервер получает значок ✓

**Файлы:**

1. **`install/database.sql`** — добавлены поля: `is_verified`, `verify_code`, `verified_at`, `verified_by`
2. **`dashboard/verify.php`** — страница верификации:
   - Пошаговая инструкция (3 шага)
   - Генерация кода, кнопка копирования
   - Пинг сервера и проверка MOTD
   - Передача владения при успехе
3. **`server.php`** — значок «✓ Владелец подтверждён» + блок «Это ваш сервер?» для незарегистрированных серверов
4. **`servers.php`** — значок ✓ в каталоге
5. **`index.php`** — `is_verified` в запросах главной
6. **`dashboard/index.php`** — кнопка «🔐 Верифицировать» / значок «✓ Владелец»

**Тесты (`VerificationTest.php`):**
- Файлы существуют, поля в БД, инструкция, генерация кода, пинг, проверка MOTD, передача владения, CSRF, значки на страницах
