# CraftRadar — Полный список функций и функционала

## 🌐 ПУБЛИЧНЫЕ СТРАНИЦЫ

| Файл | Функция | Статус |
|------|---------|--------|
| `index.php` | Главная — топ по голосам, топ по онлайну, новые серверы, статистика, поиск с автодополнением, частицы | ✅ |
| `servers.php` | Каталог — пагинация, фильтры (режим, версия, онлайн), сортировка (голоса, онлайн, рейтинг, новые), поиск | ✅ |
| `server.php` | Страница сервера — описание, статус, график 24ч/7д/30д, голосование, отзывы, жалоба, виджет, верификация | ✅ |
| `login.php` | Авторизация — логин/email + пароль, защита от брутфорса | ✅ |
| `register.php` | Регистрация — логин, email, пароль, валидация, bcrypt | ✅ |
| `logout.php` | Выход — очистка сессии | ✅ |
| `vote.php` | Голосование (AJAX) — 1 раз в 24ч, начисление баллов | ✅ |
| `review.php` | Отзыв (AJAX) — 1-5 звёзд, текст, пересчёт рейтинга | ✅ |
| `report.php` | Жалоба (AJAX) — на сервер/отзыв/пользователя | ✅ |
| `page.php` | Статические страницы — о проекте, правила, FAQ, оферта, контакты, услуги | ✅ |
| `404.php` | Страница ошибки 404 | ✅ |
